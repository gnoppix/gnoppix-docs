Chapter 6.

Gnoppix Repository

Gnoppix Software Repositories
In this chapter:
  • A Warning!
  • Gnoppix Software repositories

Gnoppix offer over 59,000 Software packages.

The topic of repositories is always a large one, and comes up frequently. It is an item which people often get wrong and confused with. Please take the time to read the information below and any references which is linked to before acting on anything.

A Warning!

If you use different repositories, it will break your system. Do not use Debian unstable, Ubuntu ppa or any other debian repos’s expect the following.

Gnoppix Software repositories

With network access, you should have the following entry present in /etc/apt/sources.list:

deb http://http.kali.org/kali kali-rolling main contrib non-free

deb https://archive.gnoppix.com/gnoppix community main

Additional line for source packages

deb-src http://http.kali.org/kali kali-rolling main contrib non-free

deb-src http://archive.gnoppix.com/gnoppix community main
