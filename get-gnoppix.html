How to get Gnoppix

How to get Gnoppix and how to make it bootable

Gnoppix Linux is supported on amd64 (x86_64/64-bit) i386 (x86/32-bit) and ARM platforms. Where possible, we would recommend using the amd64 images. The hardware requirements are minimal as listed in the documentation, although better hardware will naturally provide better performance. You should be able to use Gnoppix Linux on newer hardware with UEFI and older systems with BIOS.

Our i386 images, by default use a PAE kernel, so you can run them on systems with over 4 GB of RAM.

Generally, we release a version every 14 days, which includes bug fixes, general performance changes, and new software. For example, the release for 1st February 2023 is 23.3.

An exception is for Gnoppix Members/Supporter/Patreons who will also get .15 releases which arrive on the 15th of the month; the release looks like this: 23.2.15.
